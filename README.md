# Freaky Fashion

Detta är ett skolprojekt i kursen Backend-utveckling med JavaScript.

## Funktioner

- Startsida med produkter
- Produktsida för varje produkt
- Admin-sida
- Lägga till produkter
- Ta bort produkter
- SQLite-databas
- Express.js
- EJS-vyer
- Responsiv CSS med mobile first
